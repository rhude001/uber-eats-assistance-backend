# Uber Eats Assistance - Backend

Backend Node.js/Express pour le service d'assistance technique Uber Eats.

## Fonctionnalités
- Upload de vidéos sécurisé
- Génération de PDF automatique
- Envoi d'emails avec pièces jointes
- API REST sécurisée
